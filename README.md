# photo-trimmer

Easy photo trimming GUI tool.

## Installation

Clone this repository, and

```sh
$ yarn install
$ yarn build
$ npm link .
```

## Usage

Launch from command line.

```sh
$ photo-trimmer [src] [dest]
```

## Development

Start dev server.

```sh
$ yarn start
```

And launch electron.

```sh
$ NODE_ENV=development ./index.js [src] [dest]
```
